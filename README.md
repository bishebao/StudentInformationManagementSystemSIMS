<h2>一、项目简介</h2>
基于servlet的学生信息管理系统：前端 html、jquery、layui，后端 servlet，一个简洁的学生信息、教师信息管理系统
<h2>二、项目技术</h2>
一个echarts使用的实例系统、可以在网页上添加自己的图表、拖动图表位置进行布局
<h2>三、功能介绍</h2>
<div class="markdown-heading" dir="auto">
<div class="markdown-heading" dir="auto">基本功能：登录、退出、注册
学生信息管理：根据学号查询、学生列表、添加学生、学生信息修改、学生信息删除、学生信息导出
教师信息管理：根据姓名查询、教师列表查询、添加教师、教师信息修改、教师信息删除</div>
</div>
